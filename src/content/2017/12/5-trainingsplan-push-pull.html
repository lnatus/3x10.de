Trainingsplan - Push Pull - für mehr Muskelmasse

Beim Push Pull Training werden die Muskelgruppen grob in drückende (Push) und ziehende (Pull) Bewegungsmuster unterteilt. Teilweise sind die Grenzen hier fließend. Am erste Tag werden also alle Zugübungen ausgeführt, am zweiten Trainingstag alle Drückübungen. Es handelt sich hierbei um einen klassischen Zweier-Split, der in der Regel an vier Tagen pro Woche zum Einsatz kommt.

Für wen ist das Push Pull Training geeignet?

Sowohl für Anfänger als auch für fortgeschrittene Trainierende stellt das Push Pull Programm eine echte Option dar. Auch für sehr erfahrene Athleten die nach einem größeren Split-Programm trainieren, können mit dem Push Pull Plan eventuell neue Reize setzen.

Trainingsmuster

Folgende Trainingsmuster können zum Einsatz kommen:

  • Wochenplanung
  • Montag - Push
  • Dienstag - Pull
  • Mittwoch - Pause
  • Donnerstag - Push
  • Freitag - Pull
  • Samstag - Pause
  • Sonntag - Pause
  • Rhytmusplanung
  • Tag 1 - Push
  • Tag 2 - Pause
  • Tag 3 - Pull
  • Tag 4 - Pause
  • Start bei Tag 1

Trainingsplan im Überblick

Folgende Muskelgruppen werden im Push Plan trainiert:

  • Brust
  • Schulter
  • Trizeps
  • Beinvorderseite

Folgende Muskelgruppen werden im Pull Plan trainiert:

  • Rücken
  • Bizeps
  • Beinbeuger
  • Waden

Hinweis: Die Muskelgruppe Waden wird in den Pull-Plan integriert, da hier das Gesamtvolumen etwas geringer als beim Push Plan ausfällt. Für wen es regenerativ kein Problem sein sollte, kann dies auch gerne ans Ende des Push Tages legen. Die Muskelgruppe Bauch wird nicht gesondert aufgeführt und kann nach Bedarf in jedem Training zwischen 3 - 6 Sätze trainiert werden.
